🛒 Grocery Game Outline v2.0 — Family Edition

Main Menu (4 buttons)

  1. 🛒 Shop → Choose Child or Parent → Shared list with different permissions
  2. 🍳 Recipes — Recipe cards with ingredients, click to add all to list
  3. 💰 Budget — Add recipes, track spending vs $20 budget
  4. 📋 Lists — Export current list, future: history & store sorting

Shop Flow

  1. Click Shop → "Who's Shopping?" screen
  2. Choose 👶 Child or 👩 Parent
  3. Child view: checkboxes, add items, NO remove button
  4. Parent view: checkboxes + remove (X) button
  5. Click ➕ Add Items → 12 categories → items → back to list
