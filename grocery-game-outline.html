🛒 Grocery Game Outline v2.1

Main Menu (1 button)

🛒 Shop — That's it! Everything else flows from here.

Shop Flow

  1. Click 🛒 Shop → "Who's Shopping?" screen
  2. Choose 👶 Child or 👩 Parent

👶 Child View

  • 📋 My List (checkboxes, add items, NO delete)
  • 🍳 Simple Recipes (3-4 easy ones: PB&J, Grilled Cheese, Mac & Cheese, Fruit Salad)
  • 💰 Kid Budget (simple: "You have $20" - can't edit, just shows)
  • ➕ Add Items → categories

👩 Parent View

  • 📋 My List (checkboxes, add items, CAN delete)
  • 🍳 Full Recipes (all 6+ with prices, ingredient editing)
  • 💰 Full Budget (editable amount, tracks spent)
  • 📋 Lists → Export / Sort by Store / History
  • ➕ Add Items → categories

✅ Built

  • Single Shop button
  • Who's Shopping → choose Child or Parent
  • Child: list + checkboxes, no delete
  • Parent: list + checkboxes + delete
  • 12 categories
  • 6 recipes
  • Budget tracker
  • Lists/Export screen
